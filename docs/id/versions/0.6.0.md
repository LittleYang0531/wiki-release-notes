# 0.6.0

## Perubahan

- Penulisan ulang selesai.
- Peningkatan UI dan UX.
- Mesin audio Macaw yang baru: konsumsi memori yang lebih sedikit, offset yang lebih baik, dan mendukung lebih banyak format audio.
- Menambahkan bahasa Inggris bawaan.
- Menambahkan navigasi halaman yang lebih baik.
- Menambahkan `EffectAudio` di efek.
- Menambahkan warna _foreground_ dan _background_ yang dapat disesuaikan di gameplay UI.
- Menambahkan beberapa offset perangkat.
- Menambahkan FPS di pengaturan pengembang.
- Menambahkan sentuhan di pengaturan pengembang.
- Mengubah `Mod` untuk melakukan modulo matematika.
- Mengubah _callback_ `spawnOrder` untuk dieksekusi setelah pengoptimalan sistem.
- Mengubah sistem partikel GPU menjadi tidak aktif secara default.
- Menghapus `EffectClip` dari efek.
- Menghapus fungsi `Execute0`.
- Menghapus kemampuan _callback_ `spawnOrder` untuk menulis ke blok yang tidak dapat diubah.
- Menghapus padding bawaan untuk gameplay UI.
